# 📖 About Repository

**To-Do-List-using-ReactJS** is a modern and responsive task management web application built using React.js, Vite, and Tailwind CSS.

The application helps users organize daily tasks, manage productivity, and track task completion through a clean and interactive user interface.

This project demonstrates modern frontend development practices using component-based architecture and fast React development workflows.

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/ERAnkit03/To-Do-List-using-ReactJS.git
```

---

## 2️⃣ Navigate Into Project Folder

```bash
cd To-Do-List-using-ReactJS
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Start Development Server

```bash
npm run dev
```
